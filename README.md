# VX Futures Trading Strategy (Demo Project)

## 📌 Project Overview

This repository contains a quantitative trading strategy based on **VX (VIX Futures)**.

⚠️ **Note:**
This project is for **demonstration purposes only**.
Due to data licensing and availability, the dataset is **not included**, so the strategy cannot be executed directly.

---

## ⚙️ Environment Setup

It is recommended to use a virtual environment.

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the environment

* Mac / Linux:

```bash
source venv/bin/activate
```

* Windows:

```bash
venv\Scripts\activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```
